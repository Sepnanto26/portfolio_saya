# 📌 Personal Portfolio — Sepnanto Muresta Saputra

## 🚀 Tentang Project

Website portfolio ini dibuat sebagai media untuk memperkenalkan pengalaman, skill, project, serta sertifikasi profesional yang dimiliki. Dibangun dengan struktur modern, responsif, dan dilengkapi efek interaktif agar tetap ringan namun menarik.

## 🧩 Teknologi yang Digunakan

* **HTML5** — Struktur halaman
* **CSS3 (Responsive + Modern UI)** — Styling & layout
* **JavaScript Vanilla** — Interaktivitas & animasi
* **Canvas Animation** — Efek background interaktif
